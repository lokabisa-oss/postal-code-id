# postal-code-id

Indonesian Postal Code Dataset  
Linked to region-id administrative codes

## Disclaimer

Postal codes are operational data primarily managed by Pos Indonesia.
This repository provides an aggregated, non-authoritative dataset
for reference and interoperability purposes only.

## Relationship with region-id

- region-id: authoritative administrative region codes
- postal-code-id: postal service reference dataset

This dataset MUST NOT be used as a replacement for official
administrative region definitions.
